
# AI Resume Screening System

## Overview
This project builds an AI-powered resume screening system using LangChain.

## Features
- Skill extraction from resumes
- Matching with job description
- Scoring system (0–100)
- Explainable output

## Tech Stack
- Python
- LangChain
- HuggingFace

## Project Structure
- prompts/
- chains/
- main.py

## Output
- Strong candidate → High score
- Weak candidate → Low score

## Note
LangSmith tracing configured but limited due to API restrictions.
